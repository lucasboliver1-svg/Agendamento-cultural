##  Sobre o Projeto
Sistema web desenvolvido para digitalizar e organizar as reservas de horários do **Espaço Cultural Galpãozinho**, localizado no Gama-DF. O local atende diversos grupos da comunidade (práticas de capoeira, danças, lutas e ginástica) e o sistema substitui o antigo controle manual em papel por um catálogo digital em tempo real.

O projeto foi criado como parte da disciplina de **Projeto de Extensão II - Ação e Difusão Cultural**, do curso de Análise e Desenvolvimento de Sistemas (ADS) da instituição Anhanguera. 

**Objetivo ODS (ONU) atendido:** * **ODS 11 (Cidades e Comunidades Sustentáveis) - Meta 11.4:** Fortalecer esforços para proteger e salvaguardar o patrimônio cultural, otimizando o uso e o acesso a espaços públicos.

## Funcionalidades
- [x] Formulário de agendamento de espaços (Nome do grupo, atividade, data e horário).
- [x] Leitura e exibição em tempo real da agenda do espaço.
- [x] Bloqueio/Prevenção visual de conflitos de horários.
- [x] Interface 100% responsiva (Mobile-first), facilitando o acesso de líderes culturais via smartphone.

##  Tecnologias Utilizadas
O projeto foi desenvolvido com foco na simplicidade e eficiência, separando o front-end do back-end em nuvem:

* **Front-end:** HTML5, CSS3 (com tipografia do Google Fonts) e JavaScript (Vanilla).
* **Back-end / Banco de Dados:** Supabase (PostgreSQL). As tabelas foram estruturadas via scripts SQL diretamente na plataforma.
* **Hospedagem Front-end:** GitHub Pages.

## 💻 Estrutura do Banco de Dados (SQL)
Para o armazenamento e consumo via API no JavaScript, foi criada a seguinte tabela no Supabase:

```sql
CREATE TABLE agendamentos (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome_grupo TEXT NOT NULL,
  atividade TEXT NOT NULL,
  data_reserva DATE NOT NULL,
  horario TEXT NOT NULL,
  criado_em TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
````
